# 简介

这是一个Visual Studio Code插件，作用是实时将输入的中文标点符号自动替换为英文标点符号

github仓库地址：https://github.com/helloworldbugs/Chinese-Punctuation-Replacer

有问题欢迎提Issue

目前已支持的中英文标点符号转换映射如下

`、` -> `\`
